On a Windows PC, a device driver first needs to be installed for the module to be recognised by the Firmware Tool. To install the driver, please follow these steps:

  1.   A Windows application called Zadig will be used to install the driver. Download and install the latest version of Zadig here.

  2.   Wave Packets needs to boot in to 'firmware update mode'. Follow steps 1-5 on 'Connecting Wave Packets' below this section, and then return to step 3 here.

  3.   Assuming Wave Packets is now correctly in 'firmware update mode', open Zadig. In the menu bar, click on Options and tick 'List All Devices'.
  4.   Select "STM32 BOOTLOADER" from the dropdown menu, and select "WinUSB" to the right of the green arrow. Then press "Install Driver".
  5.   Once the driver installation is complete, the module is ready to be used with the Firmware Tool.


Connecting Wave Packets
Before the Firmware Tool can be used to issue updates, Wave Packets first needs to boot in to 'firmware update mode'.

Follow these steps strictly in this order:

  1.   Turn off your Eurorack power supply.

  2.   If you have the Wave Packets expander module, disconnect the expander data cable from Wave Packets.

  3.   Connect a USB cable between the port (USB Micro B) on the back of the module and your computer/ device.
  4.   Hold the PCB button while turning your Eurorack power supply on. Release button once power is on.
  5.   If the LEDs remain off once powered on, don't panic - this means it's successfully booted in to firmware update mode.

  6.   The module should now communicate with the Firmware Tool! The USB device name is STM32 BOOTLOADER.
